# Revenue Data and Dashboard Analysis

This project extracts and analyzes stock and revenue data for Tesla and GameStop using Python.

## Technologies Used
- Python
- yfinance
- pandas
- BeautifulSoup
- requests
- Plotly

## Project Tasks
1. Extract Tesla stock data using yfinance
2. Extract Tesla revenue data using web scraping
3. Extract GameStop stock data using yfinance
4. Extract GameStop revenue data using web scraping
5. Build Tesla dashboard
6. Build GameStop dashboard
